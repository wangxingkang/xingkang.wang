    #移动端调试

    #chrome://inspect

    在安卓设备上调试移动设备上的网页

    #启用 USB 调试

    1. 打开安卓设备的 设置。
    2. 进入 关于手机（或关于平板电脑）。
    3. 连续点击 版本号 七次，直到提示“您已处于开发者模式”。
    4. 返回 设置 主菜单，进入 系统 -> 开发者选项。
    5. 开启 USB 调试 开关。

    #连接设备

    • 使用 USB 数据线将安卓设备连接到计算机。
    • 如果设备弹出“允许 USB 调试吗？”的提示，点击 允许（建议勾选“始终允许来自此计算机的调试”）。

    #远程调试步骤

    #通过 USB 直接调试

    1. 在计算机上打开 Chrome 浏览器，访问 chrome://inspect。
    2. 确保勾选了 Discover USB devices。
    3. 等待几秒钟，你将在页面上看到连接的安卓设备以及在该设备上运行的 Chrome 标签页。
    4. 点击对应标签页下的 inspect 链接即可打开 DevTools 进行调试。

    #通过 ADB 进行无线调试 (可选)

    如果需要摆脱数据线，可以使用以下步骤：

    1. 启用端口转发

    在终端执行命令：

      adb tcpip 9222
    1. 获取设备 IP 地址

    在设备上查看：设置 -> 关于手机 -> 状态 -> IP 地址。

    1. 配置远程调试
    • 访问 chrome://inspect
    • 点击 Port forwarding 下的 Add rule。
    • 输入设备 IP 和端口（例如 192.168.1.100:9222）。
    1. 开始调试

    在设备上打开网页后，在计算机的 chrome://inspect 页面点击 inspect。

    #注意事项

    • ADB 工具：确保计算机已安装 Android SDK Platform-Tools（含 adb 命令）。
    • 防火墙：确保网络环境允许 9222 端口通信。
    • 网络连接：进行无线调试时，设备和计算机必须处于同一 Wi-Fi 网络下。
